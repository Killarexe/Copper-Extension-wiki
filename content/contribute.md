Copper Extension is an **open-source Minecraft Mod!** So anyone can contribute to it. And there is multiple ways to support this mod. But **you can also contribute to this wiki!**

## Translations

If you want to add a translation to the mod a simple pull request or an issue/ticket with the language json file added it enough for your translation to be added into the mod. You can make your pull request [here](https://github.com/Killarexe/Copper-Extension/pulls).

## Feature requests / Mod bugs

You can make simply make an issue/ticket on the [Copper Extension Github Repositories](https://github.com/Killarexe/Copper-Extension/issues). I'll surely respond at least to the next day or two.

## Coded features / Bug patches 

This one is a bit complicated. It will depends on your content of your code/edited code and pull request. But don't hesitate to at least try. I'll be really glad! You can make your pull request [here](https://github.com/Killarexe/Copper-Extension/pulls).
