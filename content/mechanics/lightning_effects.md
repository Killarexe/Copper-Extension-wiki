Lightning effects is a special feature which give for **the entity who wears copper armors to get special effects when it's srike by a thunderbolt.** This feature was **added since Copper Extension 1.5.0.**

![](lightning_effect.png)

Each oxidation level gives different effects. And each armor parts give a different duration.

Armor part and effect duration table:

| Armor part | Duration |
| ---------- | -------- |
| Helmet     | $$15s$$  |
| Chestplate | $$24s$$  |
| Leggings   | $$21s$$  |
| Boots      | $$12s$$  |


**When an copper armor part is waxed, the effect goes up by a level**. Execpt the oxidized copper armor.

Copper armor oxidation level and effect type table:

| Oxidation Level | Effect *(Normal)* | Effect *(Waxed)* |
| --------------- | ----------------- | ---------------- |
| None            | Haste             | Haste II         |
| Exposed         | Speed             | Speed II         |
| Weathered       | Resistance        | Resistance II    |
| Oxidized        | *None*            | Water Breathing  |

If you want more information on different copper armors, check [here](../copper_armors_and_tools).
